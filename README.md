# CHT-Discord-Bot
## Introduce
CHT Bot is a bot with useful tools for Hex programming and education


## Feature:
- Char to hex
- Ping
- Char to lua
- Delete trash bytes
- Hex to char
- Char to hex
- Hex to lua

## Setup
1. Install python
2. Install `py-cord` library
3. Run `python main.py`
